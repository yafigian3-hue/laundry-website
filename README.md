# 🧺 Laundry Website - Landing Page

Sebuah proyek website landing page modern dan responsif untuk layanan jasa laundry. Website ini dirancang untuk memberikan informasi layanan, paket harga, dan kemudahan bagi pelanggan untuk menghubungi penyedia jasa.

## 🚀 Demo
Lihat website yang sudah dideploy di sini: [https://yafigian3-hue.github.io/laundry-website/](https://yafigian3-hue.github.io/laundry-website/)

## ✨ Fitur Utama
- **Desain Responsif:** Tampilan yang optimal di perangkat mobile, tablet, maupun desktop.
- **Daftar Layanan:** Informasi mendetail mengenai jenis cuci yang disediakan (Cuci Kering, Cuci Setrika, dll).
- **Paket Harga:** Tabel harga yang jelas untuk memudahkan pelanggan.
- **Navigasi Modern:** Menu navigasi yang halus dan interaktif.
- **Kontak & Lokasi:** Bagian khusus untuk mempermudah pelanggan menemukan lokasi atau menghubungi via WhatsApp.

## 🛠️ Teknologi yang Digunakan
- **React.js** - Library JavaScript untuk membangun UI.
- **Tailwind CSS** - Framework CSS untuk styling yang cepat dan modern.
- **Lucide React** (Opsional, jika pakai icon) - Untuk ikon yang bersih dan ringan.
- **Vite** - Build tool untuk performa pengembangan yang cepat.
